# Event POS

เว็บแอพ POS แบบไฟล์เดียวสำหรับขายของตามงาน event ใช้บน iPad ผ่าน Safari ได้ เหมาะกับงานที่ขายเป็นครั้งคราวและไม่อยากจ่าย subscription รายเดือน

## ฟีเจอร์

- เพิ่ม/แก้ไข/ลบสินค้า พร้อมราคา SKU หมวดหมู่ และจำนวนสต็อก
- สร้างหมวดหมู่สินค้าไว้ก่อน แล้วเลือกหมวดตอนเพิ่มสินค้า
- แนบรูปสินค้าได้ พร้อมรูป placeholder ถ้าไม่ได้แนบ
- แตะสินค้าเพื่อเพิ่มลงบิล
- ปรับจำนวนสินค้าในบิล
- แสดงรูปสินค้าในหน้าขาย หน้าสต็อก และ thumbnail ในบิล
- ระบุชื่อลูกค้าและช่องทางจ่ายเงิน: เงินสดหรือโอน
- คำนวณยอดรวม ส่วนลด รับเงิน และเงินทอน
- ปิดการขายแล้วตัดสต็อกอัตโนมัติ
- ดูยอดขายวันนี้ จำนวนบิลวันนี้ และสินค้าใกล้หมด
- ดูประวัติบิลย้อนหลัง
- ส่งออกข้อมูลทั้งหมดเป็น ZIP พร้อมรูปสินค้า และนำเข้า ZIP ไปใช้บนเครื่องอื่นได้
- ทำงานแบบ static web app ไม่มี backend และไม่มีค่าสมาชิก

## วิธีเปิดบนเครื่องนี้

เปิดไฟล์ `index.html` ด้วยเบราว์เซอร์ หรือรัน local server:

```powershell
python -m http.server 4173 --bind 127.0.0.1
```

แล้วเข้า:

```text
http://127.0.0.1:4173
```

## วิธีใช้บน iPad

ทางที่ง่ายและนิ่งที่สุดคือเอาโฟลเดอร์นี้ไปวางบน static hosting ฟรี เช่น GitHub Pages, Cloudflare Pages หรือ Netlify แล้วเปิด URL นั้นบน Safari ใน iPad

จากนั้นกด Share > Add to Home Screen เพื่อให้เปิดเหมือนแอพ

## หมายเหตุเรื่องข้อมูล

ข้อมูลถูกเก็บในเครื่องของเบราว์เซอร์ด้วย `localStorage` ถ้าล้างข้อมูล Safari หรือเปลี่ยนเครื่อง ข้อมูลจะหายได้ ควรกด “ส่งออกข้อมูล” หลังจบงานเพื่อสำรองยอดขายและสต็อกไว้
